# GuildQuest-Mini-Adventure-Environment (GMAE)

Welcome to GuildQuest! This environment allows you to create and manage mini-adventures, quest events, and realms.

## How to Start a Mini-Adventure

Follow these steps to get your first adventure running:

### 1. Launch the Application
Run the `main.py` script and enter a username to log in.
```bash
python main.py
```

### 2. Create a Realm
Mini-adventures must take place within a Realm.
- From the main menu, select **(3) Realms**.
- Select **(1) Add realm**.
- Enter a name for your realm and its (x, y) coordinates.

### 3. Create a Mini-Adventure
- Return to the main menu and select **(1) Mini Adventures**.
- Select **(1) Add mini-adventure**.
- Provide a name and optional description.
- When prompted, enter the exact name of the **Realm** you created in Step 2.
- Choose an adventure type:
    - **(1) Generic**: Basic adventure structure.
    - **(2) Relic Hunt**: Competitive or Co-op mode to find relics.
    - **(3) Royal Escort**: Protect a VIP during travel.

### 4. Join and Play
- In the Mini-Adventures menu, select **(3) Join mini-adventure**.
- Type the name of the adventure you just created to start the game loop!
